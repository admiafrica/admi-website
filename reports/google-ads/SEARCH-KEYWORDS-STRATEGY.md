## üìä KEYWORD ORGANIZATION BY MATCH TYPE

### Exact Match Keywords [HIGHEST INTENT]
- "$[keyword]$" format in Google Ads
- Example: "film production diploma kenya"
- Why: Only triggers for exact searches
- Best for: High-intent, ready-to-enroll searchers
- CPC: Higher, but better quality traffic
- Expected CTR: 5-8%
- Expected Conversion Rate: 8-12%

```
"film production diploma kenya"
"music production diploma kenya"
"graphic design diploma kenya"
"photography diploma kenya"
"creative media diploma kenya"
"diploma admission open now"
"enroll diploma course nairobi"
```

### Phrase Match Keywords [MEDIUM-HIGH INTENT]
- "[keyword phrase]" format
- Example: "film production course kenya"
- Why: Includes word order, allows words before/after
- Best for: Intention is clear, flexible phrasing
- CPC: Medium
- Expected CTR: 3-5%
- Expected Conversion Rate: 5-8%

```
[enroll film production course nairobi]
[professional diploma admission kenya]
[start creative media diploma]
[apply graphic design certificate]
[become professional photographer kenya]
```

### Broad Match Keywords [LOWEST INTENT - USE SPARINGLY]
- keyword format (no quotes/brackets)
- Example: film production course
- Why: Matches variations, related terms
- Best for: Finding new keyword ideas
- CPC: Lower, but traffic quality varies
- Expected CTR: 1-2%
- Expected Conversion Rate: 0.5-2%
- **Recommendation**: AVOID for diploma programs. Use phrase match instead.
